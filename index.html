RU Direct

Blazing Fast

RU Direct is proven to be much faster than the default Rutgers App. We implemented many data structures to do just that.

Need Support?

If you ever run into a problem with the app and need help, feel free to contact us.

Minimalistic Design

We designed the app to be very minimalistic and simple to use. The user interface was kept in mind for speed!

Hack RU

RU Direct started up at Hack Rutgers University during Spring 2015.

Free!

RU Direct is completely free! There is no catch -we just wanted something faster and more convenient for Rutgers buses.

Open Source

RU Direct is open sourced on Github! You can help contribute to our app if you would like.

Features

  • Scroll

    Scroll between active buses, directions, and all Buses.

  • Simple Design

    RU Direct has a very simple layout and will stay in portrait.

  • API Use

    We use a few APIs to ensure the reliability of the app, including NextBus and Google Maps.

  • Directions and Map

    RU Direct has a map associated with each bus. We also implemented the fastest directional route to a building.

  • Android and iOS

    Currently RU Direct is only available for Android users, but we will be developing an iPhone version soon!

  • Auto Refresh

    RU direct will auto refresh every 30 seconds to keep up with buses. You can also manually refresh it.
